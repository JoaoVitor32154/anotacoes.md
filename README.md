#O Git é uma ferramenta que ajuda programadores e desenvolvedores a guardarem versões sobre todo o seu projeto, controlando suas mudanças feitas ao longo dele,
ajuda a trabalhar em equipe de forma organizada e consegue facilmente integrar contribuições de diferentes membros da equipe.

#O GitHub é uma plataforma online que utiliza o Git como base para o controle de versões, mas oferece um ambiente visual e colaborativo para armazenar,
gerenciar e compartilhar seus projetos. Ele funciona como uma espécie de “rede social” para desenvolvedores,onde é possível hospedar repositórios,
contribuir com outros projetos e acompanhar o progresso do seu código em tempo real.

#Um repositório e como se fosse uma pasta online que você pode deixar seus arquivos armazenados para acessa-los e ver seu histórico de mudança quando quiser,
dando assim uma organização com eles muito melhor.

#É um arquivo de texto simples, normalmente colocado na raiz do projeto, onde você escreve padrões de nomes de arquivos e diretórios que o Git deve ignorar.
